Privacy Policy

Single Purpose

The sole purpose of the YT+ Chrome extension is to allow users to control the playback speed of YouTube Shorts using a button or keyboard shortcut.
The extension does not collect, store, or transmit any personal data.

Data Collection

  • No personal data or browsing history is collected or transmitted.
  • All user preferences (such as selected playback speed) are stored locally in your browser using Chrome's local storage.
  • No information is shared with third parties.

Permissions Justification

  • tabs: Required to detect and interact with the currently active YouTube tab, ensuring the extension only affects YouTube Shorts.
  • storage: Used to save user preferences locally in your browser. No data is sent to external servers.
  • Host Permission (https://www.youtube.com/*): Needed to inject playback speed controls and listen for user interactions on YouTube Shorts.

All permissions are strictly necessary for the extension's single purpose and do not enable any form of data collection.
